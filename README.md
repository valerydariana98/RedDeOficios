# RedDeOficios

## 📌 Descripción

RedDeOficios es una plataforma web que conecta trabajadores de oficios 
(electricistas, plomeros, albañiles, carpinteros, entre otros) con personas 
que necesitan sus servicios.

La aplicación permite a los técnicos registrarse, crear un perfil profesional 
y ser encontrados fácilmente por usuarios que buscan servicios específicos.

---

## 🎯 Objetivo del Proyecto

Construir una aplicación web fullstack que permita:

- Registro y gestión de perfiles de técnicos
- Visualización y búsqueda de servicios
- Conexión directa entre cliente y trabajador

El proyecto está desarrollado como parte de un bootcamp fullstack, 
con enfoque en buenas prácticas, arquitectura limpia y futura escalabilidad.

---

## 🏗 Arquitectura

La aplicación sigue una arquitectura cliente-servidor:

Frontend:
- React
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

Base de Datos:
- PostgreSQL

Contenerización:
- Docker

---

## 🚀 Funcionalidades (MVP)

- Registro de técnicos
- Creación y edición de perfil
- Listado de técnicos
- Búsqueda por especialidad
- Visualización de perfil individual
- Contacto directo mediante número telefónico
